# Slash Beta Program

Welcome to the Slash Beta Program. During this beta phase, we are opening the core desktop application client and its local ecosystem.

## Beta Architecture & Limitations

1. **Local-First Runtime**: The Slash Desktop client is designed to be fully local-first. You can use it as a complete personal knowledge manager without connecting to any remote server.
2. **Slash Server Status**: The main collaborative synchronization engine (`slash-server`) is currently private as we stabilize enterprise-grade synchronization, permissions models, asset management, and secure multiplayer editing.
3. **Official Sync Service**: Users in the beta program can apply for early access to the Official Sync Service.
4. **Data Security**: Although we strive for high stability, this is beta software. Please make regular backups of your local database and asset folders.

## Feedback and Support

If you encounter any bugs or have feature requests, please open an issue in this repository or report it through our beta feedback channel.
