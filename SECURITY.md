# Security Policy

## Supported Versions

We actively support and patch security issues in the latest stable version of Slash Desktop.

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue. Instead, report it privately to our security team at `security@slash.tools`.

We will:
- Acknowledge receipt of your report within 48 hours.
- Work on a patch or mitigation strategy as quickly as possible.
- Provide a timeline for resolution and coordinate a public disclosure once a fix is released.
