# Slash 用户输入与操作指南

欢迎使用 Slash 编辑器！本文档将指导您如何使用内置的各项功能，通过 Markdown 语法和快捷键进行高效的文本输入与排版。

---

## 快速浏览 (Slash Menu)

除了手动输入 Markdown 语法外，您随时可以通过输入 `/` (斜杠) 呼出快捷菜单。目前支持：
- **Code Block / 代码块**: 快速插入多行代码块
- **Table / 表格**: 快速插入一个 3x3 表格
- **Drawing / 画板**: 快速插入一个无限画布(Tldraw)
- **AI Skills**: 快速调用您配置的各类 AI 技能进行文本处理

---

## 文本格式化 (Marks)

Slash 支持丰富的行内格式，您可以使用 Markdown 语法或快捷键触发：

| 格式 | Markdown 语法 | 快捷键 (Mac) | 说明 |
| :--- | :--- | :--- | :--- |
| **加粗** (Bold) | `**text**` | `Cmd + B` | 输入全角或半角星号均生效。在词尾按 `Backspace` 键可退回编辑模式 |
| *斜体* (Italic) | `*text*` | `Cmd + I` | 输入全角或半角星号均生效。在词尾按 `Backspace` 键可退回编辑模式 |
| ~~删除线~~ (Strike) | `~~text~~` | `Cmd + Shift + S` | 输入全角 `～～` 或半角 `~~` 均有效。词尾 `Backspace` 可编辑 |
| ==高亮== (Highlight) | `==text==` | `Cmd + Shift + H` | 用于标记重点文本。词尾 `Backspace` 可编辑 |
| `行内代码` (Code) | \`code\` | `Cmd + E` | 显示行内代码。词尾 `Backspace` 可退回 \`code 模式 |

*提示*：Slash 优化了由于中英文标点引起的光标“吞字符”问题，加粗、斜体等标记在相邻标点符号或中文后会完美触发；当光标在格式文本最右侧时，按一下 `Backspace` / `退格键`即可将渲染好的节点退回原生 Markdown 代码状态（例如 `**加粗**`），方便您继续修改内容。

## 基础块级元素 (Blocks)

### 标题 (Headings)

您可以使用 `#` 加空格快速创建标题。目前最高支持 H6。

- `# ` -> H1 (一级标题)
- `## ` -> H2 (二级标题)
- `### ` -> H3 (三级标题)
- `#### ` -> H4 (四级标题)
...以此类推至 H6。

### 列表 (Lists)

Slash 支持混合嵌套列表，您可以随意在各种列表类型间进行缩进（使用 `Tab`）和反缩进（使用 `Shift + Tab`）。

- **无序列表**: 行首输入 `- ` *或* `* `
- **有序列表**: 行首输入 `1. ` 
- **任务列表**: 行首输入 `- [ ] ` （待办） 或 `- [x] ` （完成）

*操作提示*：在空列表项中按下 `Backspace` 可退出该列表级；若有子列表，它将自动向上合并，保持结构整洁。在非列表的空行按 `Tab` 键将输入 4 个空格。

### 引用区块 (Blockquote)

通过在行首输入大于号结合空格触发：
- `> ` -> 创建引用区块

## 链接与引用 (Links & Refs)

### 普通超链接 (Links)

您可以直接输入标准 Markdown 链接：
`[Slash 官网](https://xxxx.com)` -> [Slash 官网](https://xxxx.com)
或者直接粘贴或输入完整的 URL (`http://...`, `https://...`)，它会自动转换为可点击的链接。

### 双向链接 (WikiLinks)

用于在您的个人知识库（Vault）内建立页面关联：

- 触发模式：输入 `[[` 或全角 `【【` 呼出笔记搜索菜单。
- **链接到笔记**: `[[笔记名称]]`。支持输入 `]]` 或 `】】` 快速闭合确认。
- **自定义显示名称**: `[[笔记名称|自定义文本]]` （支持半角 `|` 或全角 `｜`）
- **链接到标题章节**: `[[笔记名称#章节名]]`。在输入框内键入 `#` 即可筛选目标笔记的标题。
- **章节+别名**: `[[笔记名称#章节名|自定义文本]]`

*提示*：按住 `Cmd` / `Ctrl` 键并点击 WikiLink 即可在工作区快速跳转。光标位于链接尾部按下 `Backspace` 可还原为代码文本继续修改。

## 高级元素

### 代码块 (CodeBlocks)

由于支持大量语言（JS, TS, Python, Rust, Go, C++, SQL 等），行内直接支持语法高亮：

- **输入**: 行首输入 ` ``` ` (三个反引号) 然后可紧接语言名称（例如：` ```rust `），随后按 `Enter` 键生成代码块。
- **退出**: 当代码块为空且光标在起始位置时，按 `Backspace` 即可转化为普通段落并保留反引号代码，以便修改。

### 数学公式 (Math / LaTeX)

- **行内公式**: 使用两个美元符号包裹内容，例如：`$E = mc^2$`，编辑器会自动渲染为数学公式。公式不能与普通字符粘连紧凑。如果需要修改，把光标置于公式尾部按下 `Backspace` 即可退回 LaTeX 原文进行编辑。

### Mermaid 图表 (Mermaid Diagrams)

- **输入**: 行首输入 ` ```mermaid ` 然后按回车键。
- **使用**: 在生成的区块内输入您的 Mermaid 语法代码。失焦或完成输入后，Slash 会自动渲染出对应的流程图、时序图或关系图。渲染失败时会红字提示错误行。

### 表格 (Tables)

- **创建**: 输入 `/` 选择 `Table`，或使用快捷键 `Cmd + Shift + T`。
- **操作**: 您可以在表格节点上方出现的操作栏中点击增删行列对齐等操作。在表格内按 `Tab` 即可跳至下一单元格。

### 媒体文件 (Images, Videos, Audios)

- **直接拖拽或粘贴**: 您可以将图片（PNG/JPG/GIF）、音视频文件直接拖入编辑器中，也可以复制文件/截图后直接 `Cmd + V` 粘贴。
- **图片缩放**: 单击选中图片后，拖拽边缘的蓝色控制柄（Handle）可以调整图片大小。

### 画板 (Drawing / Tldraw)

Slash 深度集成 Tldraw，支持在 Markdown 内嵌入手绘白板。
- **创建**: 使用 `/` 命令，选择 `Drawing` 选项插入白板。
- 画板内容将在后台自动持久化为本地 PNG 预览图和 JSON 状态文件，保存无缝衔接。

| 快捷键 (Mac) | 功能说明 |
| :--- | :--- |
| `Cmd + N` | 新建笔记 |
| `Cmd + Shift + N` | 新建文件夹 |
| `Cmd + F` | 在当前笔记内查找高亮文本 |
