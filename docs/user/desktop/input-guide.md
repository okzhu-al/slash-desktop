# 输入与操作指南

> 状态：当前
> 受众：用户
> 负责人：产品
> 最后审阅：2026-05-30

本文介绍 Slash 编辑器的常用输入方式、Markdown 语法和快捷操作。

## Slash 菜单

在编辑器中输入 `/` 可以打开快捷菜单。常用入口包括：

- 代码块。
- 表格。
- 画板。
- AI Skill。

## 文本格式

| 格式 | Markdown | 快捷键 |
| --- | --- | --- |
| 加粗 | `**text**` | macOS `Cmd+B`，Windows/Linux `Ctrl+B` |
| 斜体 | `*text*` | macOS `Cmd+I`，Windows/Linux `Ctrl+I` |
| 删除线 | `~~text~~` | macOS `Cmd+Shift+S` |
| 高亮 | `==text==` | macOS `Cmd+Shift+H` |
| 行内代码 | `` `code` `` | macOS `Cmd+E` |

当光标位于格式文本末尾时，按 `Backspace` 通常可以退回 Markdown 源文本继续修改。

## 标题

在行首输入 `#` 加空格创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

最高支持到 H6。

## 列表

Slash 支持有序、无序和任务列表，也支持混合多级嵌套。

| 类型 | 输入 |
| --- | --- |
| 无序列表 | `- ` 或 `* ` |
| 有序列表 | `1. ` |
| 任务列表 | `[] ` 或 `【】 ` |

常用操作：

- `Enter`：新建下一行列表项。
- `Tab`：缩进当前列表项。
- `Shift+Tab`：降低当前列表项层级。
- 空列表项按 `Backspace`：退出当前列表。

## 任务列表

任务列表可以直接写在笔记正文中。

```markdown
[] 准备发布清单
[] 检查更新说明
```

你可以点击 checkbox 切换完成状态。任务还可以在右侧任务面板中查看和切换。

在任务行内输入两个空格可以打开任务元数据菜单，用于插入日期、人员、优先级等 chip。

## 引用

在行首输入：

```markdown
> 引用内容
```

即可创建引用块。

## 链接

普通 Markdown 链接：

```markdown
[Slash](https://example.com)
```

直接粘贴 `http://` 或 `https://` 开头的 URL，也会显示为可点击链接。

## WikiLink

WikiLink 用来连接 Vault 内的笔记。

| 输入 | 含义 |
| --- | --- |
| `[[笔记名称]]` | 链接到笔记 |
| `[[笔记名称|显示文本]]` | 使用自定义显示文本 |
| `[[笔记名称#章节]]` | 链接到笔记标题 |
| `[[笔记名称#章节|显示文本]]` | 链接标题并自定义显示文本 |

也可以输入全角 `【【` 触发 WikiLink。按住 `Cmd` 或 `Ctrl` 点击 WikiLink 可以跳转。

## 代码块

输入三个反引号加语言名：

````markdown
```rust
fn main() {}
```
````

Slash 会显示语法高亮。

## Mermaid 图表

输入：

````markdown
```mermaid
flowchart LR
  A --> B
```
````

失焦或完成输入后会渲染图表。语法错误时会显示错误提示。

## 表格

可以通过 `/` 菜单选择 `Table` 创建表格。表格内可以使用 `Tab` 在单元格间移动，也可以用表格工具栏增删行列。

## 媒体文件

你可以把图片、音视频或 PDF 拖入编辑器，也可以粘贴截图或文件。Slash 会把资源保存到 Vault 的 `assets/` 目录，并在 Markdown 中引用。

图片选中后可以拖拽控制柄调整大小。

## 画板

通过 `/` 菜单选择 `Drawing` 可以插入画板。画板适合草图、流程和自由标注。

## 常用快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Cmd/Ctrl+N` | 新建笔记 |
| `Cmd/Ctrl+Shift+N` | 新建文件夹 |
| `Cmd/Ctrl+F` | 当前笔记内查找 |
| `Cmd/Ctrl+K` | 全局搜索 |

## 输入建议

- 新内容先放入 `00_Inbox`，之后再整理到 PARA 目录。
- 长内容用标题分段，方便大纲、搜索和链接。
- 行动项直接写成任务列表，后续可在任务面板中统一查看。
- 需要建立关系时优先使用 WikiLink。
